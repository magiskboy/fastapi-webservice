fastapi-webservice
==================

Asynchronous webservice

Dependencies
~~~~~~~~~~~~

Project base on `FastAPI <https://fastapi.tiangolo.com/>`__ framework -
asynchronous web framework Using
`SQLAlchemy <https://www.sqlalchemy.org/>`__ as ORMs framework

Install and Start
~~~~~~~~~~~~~~~~~

::

    $ virtualenv venv
    $ source venv/bin/activate
    $ make
