# fastapi-webservice
Asynchronous webservice


### Dependencies

Project base on [FastAPI](https://fastapi.tiangolo.com/) framework - asynchronous web framework
Using [SQLAlchemy](https://www.sqlalchemy.org/) as ORMs framework


### Install and Start
```
$ virtualenv venv
$ source venv/bin/activate
$ make
```
